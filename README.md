# Simple Projects in Python

A collection of beginner-friendly Python projects designed to practice programming fundamentals, problem-solving, and object-oriented programming concepts.

This repository contains small but practical Python applications that demonstrate core programming skills such as user input handling, file management, randomization, abstraction, loops, conditionals, and modular design.

## Projects Included

| Project             | Description                                                      | Concepts                                   |
| ------------------- | ---------------------------------------------------------------- | ------------------------------------------ |
| File Organizer      | Automatically categorizes files into folders based on file type. | File Handling, Path Management, Automation |
| Password Generator  | Generates random letter, numeric, or mixed passwords.            | OOP, Abstraction, Randomization            |
| Rock Paper Scissors | Classic command-line game against the computer.                  | Functions, Loops, Conditionals             |
| More Projects       | Additional beginner Python projects and exercises.               | Python Fundamentals                        |

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/husseiny8/Simple_Project_in_Python.git
cd Simple_Project_in_Python
```

### Run a Project

Navigate to the desired project folder and execute the Python file:

```bash
python project_name.py
```

Example:

```bash
python password_generator.py
```
